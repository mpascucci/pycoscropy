Detailed Description

template<typename T>
class deconvolve::FFTWPlan< T >

Wrapper around FFTW plan for real-to-comlpex and complex-to-real in-place 3D transformations.

This wrapper can keep either forward or inverse FFTW plan. The plan is allocated using forward or inverse methods. After allocation, the plan can be used using execute method. On destruction or calling clean, the allocated plan is destroyed.
