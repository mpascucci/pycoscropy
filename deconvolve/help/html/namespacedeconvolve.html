typedef int(* deconvolve::callback_type) (size_t iteration_number, double min, double max, double sum, double nrm2_prev, double nrm2_prevprev, double lambda, double lambda_factor, double snr)

Callback function type used to communicate during deconvolution. For use from C.

Callback function is called during each iteration and should return zero if the iteration should be stopped. To continue iteration, return non-zero value, for example 1.

Note that the callback function is called first before the first iteration.

Function arguments:

Parameters
iteration_numberCurrent iteration (from zero onwards)
minminimal value of deconvolved image. Set for iterations with iteration_number 1 and larger
maxmaximal value of deconvolved image. Set for iterations with iteration_number 1 and larger
sumsum of all values of deconvolved image. Set for iterations with iteration_number 1 and larger
nrm2_prevEuclidean norm between current and previous estimate of deconvolved image. Set for iterations with iteration_number 1 and larger
nrm2_prevprevEuclidean norm between current and estimate from two steps before of deconvolved image. Set for iterations with iteration_number 2 and larger
lambdacurrent lambda value. Set for iterations with iteration_number 1 and larger
lambda_factorfactor used in the calculations of lambda. Corresponds to "C" in equation 5 of https://doi.org/10.1111/j.1365-2818.2011.03486.x .Set for iterations with iteration_number 1 and larger
snrused SNR estimate for original image. If not set by user, SNR is calculated before iterations. Set for all calls.
Returns
Zero to stop iterations, continue iterations for non-zero values

typedef int(* deconvolve::callback_extended_type) (void *user_data, size_t iteration_number, double min, double max, double sum, double nrm2_prev, double nrm2_prevprev, double lambda, double lambda_factor, double snr)

Extended callback function type used to communicate during deconvolution.

This is an extension of callback_type to include user-supplied data pointer that is passed on every call. All other parameters and return values are the same as for callback_type
